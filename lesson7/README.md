Lesson 9; Tìm hiểu Go Formatting Verbs
Go Formatting Verbs (còn gọi là Printf Verbs hay Format Specifiers) được dùng trong các hàm Printf, Sprintf để kiểm soát cách các giá trị được định dạng và in ra dưới dạng chuỗi. Hoạt động như các placeholder.
- %v: Dùng để lấy giá trị của 1 biến
- %T: Dùng để hiển thị kiểu dữ liệu 1 biến
- %s: String
- %d: Integer (decimal)
- %.f: Floating-point number
- %t: Boolean
- %c: Character (rune)
- %p: Pointer
- %%: Dùng khi cần hiển thị dấu %
